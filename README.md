# Task-Salary-Prediction
Salary Prediction using Linear Regression in Python. This project builds a machine learning model to predict employee salary based on experience. It uses Pandas for data handling and Scikit-learn for model training and evaluation with Mean Squared Error (MSE).
Salary Prediction using Linear Regression

This project is a simple Machine Learning model built using Python to predict employee salaries based on years of experience. It uses Linear Regression to learn the relationship between experience and salary and make predictions.

The dataset is created manually for learning purposes and contains two columns: Experience and Salary. The model is trained using Scikit-learn and evaluated using Mean Squared Error (MSE).

This project helps in understanding the basics of regression, model training, prediction, and evaluation in machine learning.

Tools Used -
Python
Pandas
Scikit-learn (Linear Regression)

Objective -
To predict salary based on experience using a simple machine learning model.

Output -
The model successfully predicts salaries close to actual values with low error.
